A beautiful web application for visualizing 3D molecular structures of organic chemistry compounds.

## Features

- **Interactive 3D Visualization**: Rotate, zoom, and explore molecules in real-time
- **Extensive Molecule Library**: 
  - Alkanes (Methane to Octane)
  - Alkenes (Ethene, Propene, Butene)
  - Alkynes (Ethyne, Propyne)
  - Cyclic compounds (Cyclopropane, Cyclohexane, Benzene)
  - Functional groups (Ethanol, Acetone, Acetic Acid, Methylamine)

- **Display Options**:
  - Ball & Stick model
  - Space-filling model
  - Atom labels toggle
  - Auto-rotation toggle

- **Beautiful UI**: Modern gradient design with smooth animations

## How to Use

1. Open `index.html` in a web browser
2. Select a compound from the sidebar
3. Use your mouse to:
   - **Drag** to rotate the molecule
   - **Scroll** to zoom in/out
4. Toggle display options as desired

## Technical Details

- Built with **Three.js** for 3D rendering
- Pure HTML, CSS, and JavaScript
- No build process required
- Responsive design
